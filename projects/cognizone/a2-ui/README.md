# @cognizone/a2-ui

Angular UI component library built on Angular Material, providing confirmation dialogs, alert notifications, and loading indicators.
Requires:
**Angular >= 20 for v1.1.0**
**Angular >= 21 for v2.0.0**

## Installation

```bash
npm i @cognizone/a2-ui
```

## Key Features

- **Are You Sure Modal**: Confirmation dialog component with customizable header, text, and action buttons
- **Global Alert Service**: Snackbar notification service with error, warning, info, success, and debug variants
- **Loading Stairs**: Loading indicator component with optional message display and dynamic updates
